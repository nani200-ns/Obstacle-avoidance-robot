An obstacle avoiding robot is an intelligent embedded system designed to detect and avoid obstacles in its path automatically.
This project is implemented using an Arduino microcontroller and an ultrasonic sensor to measure the distance between the robot and nearby objects. 
When an obstacle is detected within a predefined range, the robot changes its direction to avoid collision.
The system uses DC motors controlled through a motor driver to move the robot in different directions.
The ultrasonic sensor continuously sends and receives sound waves to identify obstacles in real time.
Based on the sensor data, the Arduino processes the information and makes decisions to stop, turn left, or turn right.
This project demonstrates the basic concepts of robotics, automation, and embedded systems.
It is useful for understanding real-time applications such as autonomous vehicles, industrial automation, and smart robotic systems.
The obstacle avoiding robot is simple, cost-effective, and suitable for academic miniprojects and learning purposes.
